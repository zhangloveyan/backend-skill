# 任务文档模板

```markdown
# 开发任务

## 当前任务

### {日期} {功能名称}

**方案文档：** `docs/方案/{功能名称}.md`（如有）

**任务清单：**

- [ ] 任务1
- [~] 任务2（进行中）
- [x] 任务3（已完成）

**提交记录：**

- `{commit-hash}` {type}: {message}

---

## 历史任务

（已完成的任务归档到这里）
```
